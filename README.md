## À propos

60 jours de formation structurée pour maîtriser le CCNA 200-301. Chaque jour correspond à un module de cours complet, lisible directement dans le navigateur, sans compte ni connexion.

**37 PDFs source → 60 pages HTML interactives**

## Structure du projet

```
ccna-60j/
├── index.html          ← Page d'accueil de la plateforme
├── jours/
│   ├── jour-01.html    ✅ Disponible
│   ├── jour-02.html    🔜 En cours
│   └── ...
└── README.md
```

## Programme

| Module | Jours | Contenu |
|--------|-------|---------|
| 01 | 1–10 | Fondamentaux Réseau — OSI, TCP/IP, Câbles |
| 02 | 11–22 | Routage IP — OSPF, EIGRP, RIP |
| 03 | 23–38 | WAN & Technologies avancées |
| 04 | 39–52 | Sécurité & Administration |
| 05 | 53–60 | Révision & Examens blancs |

## Utilisation

Clonez le dépôt et ouvrez `index.html` dans votre navigateur — c'est tout.

```bash
git clone https://github.com/votre-username/ccna-60j.git
cd ccna-60j
open index.html
```

Ou consultez directement via **GitHub Pages**.
